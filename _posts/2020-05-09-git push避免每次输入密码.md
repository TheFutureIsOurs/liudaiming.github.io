	在git push时每次都要输入用户名和密码是一件很痛苦的事
    解决方法如下：
	
<!--more-->


----------------

1.git bash进入当前项目（windows机在当前项目中点击右键->选择git bash here打开git bash）

2.在git bash中输入：

	git config  credential.helper store

3.git push到远程仓库时，输入正确用户名和密码，以后再次提交时就不用输入了。
